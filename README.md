# Weather App с мониторингом

Простое веб-приложение, отображающее текущую температуру, упакованное в Docker-контейнер и оснащённое базовым мониторингом доступности по HTTP.

---

## Используемый стек

- Python (Flask)
- Docker, Docker Compose
- Prometheus + Blackbox Exporter + Grafana (HTTP-мониторинг)

---
## Описание приложения

Приложение получает данные о текущей температуре из публичного погодного API и отображает их через HTTP-интерфейс.

Доступные эндпоинты:

- `/` — отображение текущей температуры
- `/health` — эндпоинт проверки работоспособности приложения

---
## Инструкция по запуску

1. Клонировать репозиторий:

```bash
git clone https://github.com/madnoxx/Weather-app.git
cd Weather-app
```

2. Установить переменную окружения с API-ключом:
```bash
export WEATHER_API_KEY=your_api_key
```

3. Собрать и запустить сервисы (в режиме detached):
```bash
docker compose up -d
```
